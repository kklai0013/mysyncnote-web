# MySyncNote

MySyncNote 是一個直接編輯本機 Markdown 資料夾的離線優先 PWA。筆記保持為一般的 `.md`、附件與相容 JSON Canvas 的 `.canvas` 檔案；Google Drive 同步交給 FolderSync 或其他資料夾同步工具。

正式網址：<https://kklai0013.github.io/mysyncnote-web/>

## 第一次使用

1. 使用最新版 Chrome 或 Edge 開啟正式網址。
2. 按「開啟筆記庫」。
3. 選擇存放 Markdown 筆記的資料夾並允許讀寫。
4. 之後會自動還原上次的筆記、選取資料夾、分頁、圖譜與窗格布局。若手機顯示「重新連線」，只需點一次並重新允許原資料夾。

## FolderSync

MySyncNote 不登入 Google 帳號，也不使用 Google API。請在 FolderSync 建立資料夾配對，將 Google Drive 上的筆記資料夾同步到手機本機，再讓 MySyncNote 開啟同一個本機資料夾。

MySyncNote 回到前景時會重新掃描筆記庫。若正在編輯的檔案被 FolderSync 改動，會要求選擇保留目前版本、載入外部版本或保留兩份。

## 主要功能

- 巢狀資料夾、筆記、Canvas 的新增、改名、移動、刪除與垃圾桶
- 自動儲存、外部修改偵測、分頁、搜尋、命令面板
- 單一編輯面的 Markdown 混合即時預覽、跨行選取、復原／重做、`[[` 連結與 `#標籤` 建議、原始碼、閱讀與並排預覽
- Wiki Link、自動完成、嵌入、反向連結、未連結提及、標籤與大綱
- 全筆記庫與局部關聯圖譜、資料夾／標籤篩選、孤立及缺失筆記
- 分頁拖曳排序；筆記可向上、下、左、右建立可縮放的巢狀分割窗格
- 關聯圖譜可停靠在筆記旁邊，同時閱讀、編輯與查看關係
- 相容 JSON Canvas 的文字、筆記與網頁卡片；圓點拉線、框選、Shift 多選、群組連動、顏色、標籤、復原／重做、平移與縮放
- 桌面和手機自適應介面、PWA 安裝與離線程式介面
- 自動回到上次使用位置，不需要維護「預設資料夾」設定；所有快捷鍵都可重新指定
- 可自行選擇全域 `mysyncnote-settings.json` 的實際存放資料夾，切換筆記庫不必重設

## 快捷鍵

- `Ctrl+S`：立即儲存
- `Ctrl+P`：命令面板
- `Ctrl+K`：搜尋筆記庫
- `Ctrl+N`：新增筆記
- `Ctrl+G`：關聯圖譜
- `Ctrl+Z`：復原編輯
- `Ctrl+Y` 或 `Ctrl+Shift+Z`：重做編輯
- `F2`：重新命名目前選取的檔案或資料夾
- `Ctrl+\`：向右分割目前筆記

所有快捷鍵都可以在「設定 → 快捷鍵」重新指定。

## 多窗格

- 拖曳上方分頁到窗格的左、右、上、下邊緣，可以在該方向建立分割。
- 按右上角「分割」可明確選擇向右、向下、向左或向上分割。
- 拖曳窗格標題可移動窗格；拖曳分隔線可調整寬度或高度。
- 關聯圖譜會停靠在同一個工作區，點擊圖譜節點只會更換主要筆記，不會關閉圖譜。
- 分割布局會隨筆記庫保存，下次開啟時自動還原。
- 每個分割窗格各自處理連結跳轉；從右側筆記點連結不會錯誤地改掉左側筆記。

## 全域設定檔

在「設定 → 全域設定檔」用系統資料夾選擇器選擇實際位置後，MySyncNote 會在那裡寫入 `mysyncnote-settings.json`。這個位置由所有筆記庫共用，內容包含快捷鍵及其他偏好，但不包含瀏覽器授權或帳號資料。

設定檔即使放在筆記庫裡，也不會顯示在左側筆記檔案樹。

瀏覽器安全規則不允許網頁從文字欄位直接讀寫 `C:\...` 或 Android 絕對路徑，也不會把完整系統路徑交給網頁；因此必須由使用者在系統選擇器選一次。選好後會記住該資料夾控制代碼，不會因切換筆記庫而要求重新設定。
